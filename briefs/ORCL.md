## 1. Company snapshot

Oracle เป็นบริษัทซอฟต์แวร์และ cloud infrastructure สำหรับองค์กรขนาดใหญ่ทั่วโลก จุดแข็งหลักคือ Oracle Database ที่ embed อยู่ใน core systems ขององค์กรมาหลายสิบปีและเปลี่ยนยากมาก ปัจจุบันกำลังสร้าง OCI (Oracle Cloud Infrastructure) แข่งกับ AWS/Azure/GCP และขาย cloud applications (Fusion ERP, NetSuite) ให้องค์กรที่ต้องการระบบ HR/finance/supply chain บน cloud ลูกค้าหลักคือ enterprise ทั่วโลก — cloud (IaaS + SaaS) เพิ่งข้าม 51% ของรายได้รวมเป็นครั้งแรกใน FY2026

## 2. Fundamentals signal (จาก 10-K FY2026)

- **Revenue acceleration ชัดเจน:** FY2024 $53.0B → FY2025 $57.4B (+8%) → FY2026 $67.4B (+17%) — OCI เป็นตัวขับเคลื่อนหลัก (source: 10-k-latest.md)
- **Margin ทรงดีแม้ลงทุนหนัก:** GAAP operating margin ~30% ทั้ง FY2025 (30.8%) และ FY2026 (30.6%) net margin FY2026 ขยับขึ้นเป็น 25.4% จาก 19.8% ใน FY2024 (source: 10-k-latest.md)
- **หนี้หนักมาก:** total debt $92.6B ณ FY2025 (current $7.3B + long-term $85.3B) vs. cash $10.8B stockholders' equity $20.97B — Oracle กู้หนักเพื่อ fund datacenter expansion (source: 10-k-latest.md)
- **FCF ติดลบจาก capex มหาศาล:** operating cash flow record $32.0B แต่ capex FY2026 $55.7B (จาก $21.2B ใน FY2025) ทำให้ FCF -$23.7B Oracle ระดมหนี้ $43B + equity $5B เพิ่มในปีเดียว (source: 10-k-latest.md)
- **Capital allocation เน้นลงทุน ไม่ buyback:** buyback FY2025 แค่ $600M dividends $0.50/share/quarter pattern บอกว่า management เชื่อว่า cloud capacity คือ return ที่ดีที่สุดตอนนี้ (source: 10-k-latest.md)

## 4. Bull case / Bear case

**Bull case:**
- **Switching cost จาก Oracle Database แข็งมาก:** องค์กรที่ run mission-critical systems บน Oracle Database มาหลายสิบปีไม่สามารถเปลี่ยน vendor ได้ง่ายๆ — migration cost สูงจนแทบไม่เกิดขึ้นจริง ทำให้ license support revenue มี predictability สูง และเป็นฐาน upsell ไป OCI
- **RPO $638B บ่งชี้ demand จริง:** contracts ที่ enterprise signed แล้วกับ OCI เป็นหลักฐานว่า demand ไม่ใช่แค่ pipeline — GPU utilization 97.5% ยืนยันว่าความต้องการมีอยู่จริงก่อน capacity ใหม่เปิด
- **AI demand acceleration มีฐานรองรับ:** Cloud infra +93% YoY AI infra contracts Q4 $67B management รายงาน ROIC ที่ steady state ระดับ high 20s% — ถ้าตัวเลขนี้ถูก capex ที่ลงทุนไปคุ้มค่า

**Bear case:**
- **Capex FY2027 $70B ใหญ่กว่า operating cash flow ทั้งปี — และ reverse ไม่ได้:** Oracle ต้องหา external financing อย่างน้อย $38B เพื่อปิด gap บน leverage ที่หนักอยู่แล้ว ($92.6B debt) datacenter asset เปลี่ยน repurpose ไม่ได้ในระยะสั้น ถ้า AI demand ชะลอก่อน capacity ใหม่เต็ม margin จะพัง ไม่ใช่แค่ FCF ติดลบ
- **RPO $638B ไม่มีข้อมูล recognition timeline:** enterprise contracts มักเป็น 5-10 ปี revenue ที่จะ recognize ใน FY2027 อาจแค่ส่วนหนึ่ง ถ้า 3-4 counterparty ใหญ่คิดเป็น 30%+ ของ RPO การ renegotiate ดีลเดียวทำตัวเลขหายไปมากใน quarter เดียว
- **OCI โตจาก overflow demand ไม่ใช่ market share จริง:** AWS/Azure/GCP กำลัง deploy capex $60-80B+ ต่อราย เมื่อ hyperscaler ขยาย capacity นักพัฒนา AI อาจกลับ ecosystem เดิม OCI ไม่มี proprietary AI model หรือ developer ecosystem เทียบ AWS/Azure และไม่มีหลักฐานในข้อมูลที่มีว่า OCI คือตัวเลือกแรก — แค่ตัวเลือกที่มีให้

## 5. Kill conditions

1. **OCI cloud infrastructure growth ต่ำกว่า 40% YoY สองไตรมาสติด ในขณะที่ AWS/Azure รายงาน GPU capacity เปิดเพิ่ม** — ถ้าเกิดพร้อมกันแปลว่า OCI กำลังเสีย overflow demand จริงๆ ไม่ใช่ AI demand global ชะลอ ดู: Q1/Q2 FY2027 earnings
2. **Short-term RPO (12-month portion) ไม่โตหรือโตช้ากว่า total RPO สองไตรมาสติด** — ถ้า total RPO ขึ้นแต่ short-term stagnate แปลว่า backlog ยาวขึ้น revenue จะช้ากว่าที่ตลาดคาด ดู: 10-Q filing
3. **Operating cash flow FY2027 ต่ำกว่า $38B** — เส้น minimum ที่ Oracle จะไม่ต้องระดมทุนเพิ่มก้อนใหญ่อีกรอบในปี $70B capex ดู: FY2027 annual results
4. **Oracle ตอบคำถาม capex flexibility ใน earnings call แบบหลบเลี่ยงหรือบอก "committed" โดยไม่ระบุ exit options** — ถ้าตอบแบบนี้แปลว่า capex เป็น irreversible take-or-pay ดู: Q1/Q2 FY2027 call transcript
5. **NVIDIA datacenter segment growth ต่ำกว่า 25% YoY สองไตรมาสติด** — NVIDIA เป็น leading indicator ของ AI infrastructure demand Oracle ไม่มี revenue stream อื่นรองรับเหมือน AWS (e-commerce) หรือ Azure (Office 365) ดู: NVDA earnings

## 6. What to ask before owning it

1. Capex $70B FY2027 มีส่วนไหน take-or-pay ที่ยกเลิกไม่ได้ — ถ้า AI demand ชะลอ Oracle cut capex ได้ไหม หรือต้องจ่ายอยู่ดี
2. RPO $638B กี่ % จะ recognize เป็นรายได้ใน 12 เดือนข้างหน้า — ตัวเลขนี้ใหญ่มากแต่ถ้า recognition period 5-10 ปี revenue growth อาจช้ากว่าที่คิด
3. OCI ได้ market share จาก AWS/Azure จริงหรือแค่รับ overflow — ถ้าแค่ overflow demand จะหายเมื่อ hyperscaler ขยาย capacity ใน 1-2 ปีข้างหน้า
4. Litigation investigation ที่ shareholders เริ่มเรื่อง AI capex disclosure มีโอกาสไปถึงไหน — มีผลต่อ management credibility หรือเปล่า
5. Oracle Database ยังเป็น mandatory สำหรับลูกค้าที่ย้ายไป cloud เจ้าอื่นไหม หรือ AWS RDS / Azure SQL Database เป็น alternative ที่ดีพอแล้ว
