### 4. Bull case / Bear case

**Bull case:**
- Microsoft anchor + NVIDIA co-investor: NVIDIA vest $2.1B เมื่อ deploy 600,000 GPUs บน IREN campus — ถ้า NVIDIA ยังอยู่ IREN ไม่ใช่แค่ wishful thinking *(แต่ brief ไม่มีข้อมูล deploy ไปถึงกี่ GPUs แล้วและ timeline ชัดเจน)*
- Power capacity เป็น cost advantage จาก scale economies: 2,910MW contracted grid power ใน market ที่ power scarcity คือ bottleneck ของ AI infrastructure ทั้งอุตสาหกรรม สร้างใหม่แข่งไม่ได้ใน 1-2 ปี
- AI Cloud pivot เกิดขึ้นจริงในตัวเลข: $3.1M (FY2024) → $16.4M (FY2025) → $33.6M/quarter (Q3 FY2026) — direction ถูก แต่ scale ยังห่างจาก target มาก

**Bear case (Rey):**
- BTC คือตัวแปรที่ควบคุมไม่ได้: Q3 revenue ลดจาก $184.7M → $144.8M และ EBITDA ลดจาก $75.3M → $59.5M ในไตรมาสเดียว สาเหตุคือ BTC price ไม่ใช่ execution ถ้า BTC ร่วง 50% mining revenue ไม่ cover capex $1.38B และ cash $2.6B drain เร็วกว่าที่ AI Cloud จะ ramp ทัน
- Gap ~28x *(calculated)* ระหว่าง actual กับ target: AI Cloud run rate ~$134M annualized *(calculated)* vs $3.7B ARR target ภายใน 6 เดือน contracted ARR $3.1B ≠ recognized revenue — ยังไม่ทราบว่ามี milestone/condition ซ่อนอยู่กี่ชั้นในแต่ละสัญญา
- โครงสร้างหนี้ไม่ match กับ revenue speed: capex/revenue 2.75x *(calculated)*, convertible notes $962.8M ไม่ทราบ maturity date ใน brief นี้ ถ้า notes due ก่อน AI Cloud generate cash พอ ต้อง dilute shareholders หรือ default — blind spot ที่อันตราย
