## อ่านง่าย

ASML เป็นบริษัทดัตช์ที่ขาย "เครื่องพิมพ์ลายวงจรลงชิป" — เปรียบง่ายๆ คือเป็นเครื่องจักรที่โรงงานชิปทั่วโลกต้องมี ถึงจะผลิตชิปสมัยใหม่ได้ และ ASML เป็นเจ้าเดียวในโลกที่ทำเครื่องรุ่นทันสมัยสุด (EUV) ได้ ไตรมาสล่าสุดออกมาแข็งแรงและบริษัทขึ้นเป้าทั้งปี แต่มีจุดที่ต้องระวังเรื่อง "การมองเห็นดีมานด์รอบหน้า" อยู่

**ทำอะไร:** ขายเครื่องจักรที่โรงงานชิปใช้พิมพ์ลายวงจรลงบนชิป ลูกค้าคือโรงงานชิปยักษ์ใหญ่ (TSMC, Samsung, Intel) รายได้มาจากการขายเครื่อง บวกกับค่าบริการดูแลเครื่องเก่าที่เข้ามาเรื่อยๆ ทุกปี

**ไตรมาสล่าสุด (Q2 2026):** ยอดขายและกำไรออกมาดีกว่าที่ตลาดคาด ขายเครื่องไป 86 เครื่อง และบริษัทกล้าขึ้นเป้ารายได้ทั้งปีขึ้นอีกก้อนใหญ่ เพราะดีมานด์ชิป AI แรงมาก

**จุดแข็ง:** เป็นเจ้าเดียวในโลกที่ทำเครื่อง EUV ได้ (คู่แข่งลอกไม่ได้ ต้องมาซื้อจากที่นี่เท่านั้น) งบการเงินแข็ง เงินสดเยอะ และรายได้ค่าบริการดูแลเครื่องก็โตต่อเนื่องเกิน 30% ซึ่งเป็นรายได้ที่ค่อนข้างสม่ำเสมอ

**จุดเสี่ยงที่ต้องระวัง:**
- ปีนี้ ASML เลิกเปิดตัวเลข "ออเดอร์ใหม่" ซึ่งเป็นตัวเลขที่คนภายนอกเคยใช้เช็คว่าดีมานด์รอบหน้าจะดีไหม — พอเลิกเปิดจังหวะเดียวกับที่ขึ้นเป้าพอดี ทำให้เราตรวจสอบเองไม่ได้ ต้องเชื่อคำพูดบริษัทล้วนๆ
- ลูกค้ากระจุกอยู่แค่ 3 รายใหญ่ ถ้ารายใดชะลอการซื้อ กระทบเยอะ
- ยอดขายส่วนหนึ่งเป็นชิปประเภท memory ซึ่งเป็นธุรกิจขึ้นแรงลงแรง และจีน (~20% ของยอดขาย) มีความเสี่ยงจากกฎห้ามส่งออกของสหรัฐฯ

**สรุปสั้นๆ:** โดยรวมเป็นบริษัทที่มีข้อได้เปรียบชัดมาก (ผูกขาดของที่คนอื่นทำไม่ได้) และไตรมาสล่าสุดตัวเลขดีจริง แต่จุดที่ต้องคิดคือ ตอนนี้เราตรวจสอบดีมานด์รอบหน้าเองได้ยากขึ้น เพราะบริษัทหยุดเปิดตัวเลขสำคัญ — ก่อนตัดสินใจอะไร ควรอ่าน thesis เต็มด้านล่างเพื่อดูว่าความเสี่ยงพวกนี้หนักแค่ไหนสำหรับเรา (บล็อกนี้เป็นสรุปเพื่อความเข้าใจ ไม่ใช่คำแนะนำซื้อขาย)

## 1. Company snapshot

ASML (ดัตช์, ตั้งอยู่ที่ Veldhoven) เป็นผู้ผลิตเครื่อง lithography ที่ใช้พิมพ์ลายวงจรลงบนชิป และเป็นผู้ผลิตเครื่อง EUV (extreme ultraviolet) รายเดียวในโลก ลูกค้าคือโรงงานผลิตชิปรายใหญ่ — TSMC, Samsung, Intel รายได้มาจาก 2 ทาง: ขายเครื่องใหม่ (ก้อนใหญ่ต่อเครื่อง) และ Installed Base Management (บริการ อัปเกรด อะไหล่ ของเครื่องที่ติดตั้งไปแล้ว — เป็นรายได้ที่ recurring กว่า) รายงานเป็นเงินยูโร (EUR) และเทรดในสหรัฐฯ เป็น ADR ดังนั้นมูลค่าเป็นดอลลาร์จะแกว่งตามอัตราแลกเปลี่ยน EUR/USD ด้วย

## 4. Bull case / Bear case

**Bull case**
- **ผูกขาด EUV จริง** — เป็นผู้ผลิตรายเดียวในโลก ลูกค้าที่จะทำ leading-edge node ไม่มีทางเลือกอื่น เป็น moat จาก technology + scale economies (ต้นทุน R&D + ecosystem ที่คู่แข่งสร้างตามไม่ทัน)
- **AI ดันดีมานด์ leading-edge logic + DRAM** ทำให้ต้องใช้ lithography มากขึ้น จน ASML ขึ้น guidance ปี 2026 สองรอบ (หมายเหตุ: "AI เป็นสาเหตุ" เป็น management claim — ไม่มีตัวเลขแยกว่าส่วนไหนมาจาก AI จริง vs. รอบ memory ปกติ)
- **Installed Base Management โตเร็ว** (>30% guide ปี 2026) เป็นรายได้ recurring ที่ margin ดีและสม่ำเสมอกว่าการขายเครื่อง

**Bear case** (จาก Rey)
- **ASML หยุดเปิดเผย net bookings ตั้งแต่ Q1 2026 พอดีจังหวะที่ทุกอย่างกำลัง "ดีเกินไป":** bookings คือ leading indicator เดียวที่บอกว่าดีมานด์รอบหน้าจริงหรือไม่ — บริษัทเลือกปิดมันในไตรมาสเดียวกับที่ขึ้น guidance สองรอบ (€36–40B → €43–45B) ทำให้ตอนนี้ verify คำว่า "demand extremely strong" ไม่ได้เลย ต้องเชื่อ management ล้วนๆ; backlog ล่าสุดก็ไม่เปิดเผยใน Q2 — visibility ที่เคยเป็นจุดขายของหุ้นนี้กำลังถูกปิดไฟทีละดวง
- **Guidance ที่ขึ้น +16% กลางปี เพิ่มความเสี่ยง "ตกจากที่สูง":** memory guide โต >75% ปีเดียวเป็นตัวเลขที่ร้อนผิดปกติ — memory เป็นธุรกิจ cyclical จัด (ขึ้นแรงลงแรง ไม่โตเส้นตรง) ถ้ารอบ memory พีคเร็ว (ลูกค้า double-order ตอนของขาด แล้ว cancel ตอนของล้น) FY2027 จะเจอ base สูงที่ทำซ้ำไม่ได้ การขึ้น guide กลางปีแปลว่าราคาหุ้น price-in ความสมบูรณ์แบบไว้แล้ว downside เวลา miss จึงเจ็บกว่าเดิม
- **ลูกค้ากระจุกตัวสุดขั้ว + capex ลูกค้าอยู่นอกการควบคุม ASML:** รายได้เกือบทั้งหมดมาจาก 3 ราย (TSMC, Samsung, Intel) การขึ้น guide คือการเดิมพันว่า 3 รายนี้จะไม่ลด/เลื่อน capex — สิ่งที่ ASML คุมไม่ได้ (backlog นี้ลูกค้า push out ได้ และเคยทำใน down-cycle) นอกจากนี้ China ~20% ของยอดขาย 2026 ขึ้นกับ export control สหรัฐ-เนเธอร์แลนด์ ถ้าถูกตัด = หาย 1 ใน 5 ของรายได้ทันที และเป็นดีมานด์ mainstream logic ที่หาลูกค้าอื่นทดแทนยากในระยะสั้น

*(หมายเหตุ: bear case เรื่อง High-NA ยังไม่ proven อ่อนลงในไตรมาสนี้ — Intel ขึ้น Panther Lake บน Intel 18A ด้วย High-NA เข้า production แล้ว ซึ่งเป็น mass production ไม่ใช่ R&D แต่ยังเป็นลูกค้ารายเดียว โหนดเดียว จึงย้ายไปเป็น kill condition แทน)*

## 5. Kill conditions (จาก Rey)

- **Bookings ยังมืด + guidance เริ่มถอย:** ถ้า ASML **ยังไม่กลับมาเปิด net bookings** ภายในสิ้นปี 2026 **และ** ไตรมาสใดหั่น FY2026 guidance ต่ำกว่ากรอบ €43–45B หรือ guide FY2027 ต่ำกว่า consensus ตอนนั้น → thesis "demand accelerating" verify ไม่ได้และเริ่มพัง *(monitor: press release ทุกไตรมาส — มีตัวเลข bookings ไหม + FY guide ขยับทางไหน)*
- **Memory rollover:** ถ้าไตรมาสใด memory system sales หด QoQ **2 ไตรมาสติดกัน** หรือ management พูดถึง "push-out / digestion / softening" ฝั่ง memory ใน call → รอบ +75% กำลังจบและ base ปี 2027 ทำซ้ำไม่ได้ *(baseline Q2 2026: Logic ~51% / Memory ~49%)*
- **China ถูกตัด:** ถ้า export control ใหม่ (สหรัฐหรือเนเธอร์แลนด์) บังคับใช้จริงกับเครื่องที่ขายจีนได้อยู่ **หรือ** สัดส่วนรายได้จีนร่วงต่ำกว่า ~12% ของยอดขายรวมจากฐาน ~20% โดยไม่มีลูกค้าอื่นชดเชย → หาย 1 ใน 5 ของรายได้ *(monitor: geographic split ต่อไตรมาส + ข่าว export rule ที่บังคับใช้จริง ไม่ใช่แค่จ่อ)*
- **High-NA หยุดที่ลูกค้ารายเดียว:** ถ้าถึงสิ้นปี 2027 **ยังไม่มีลูกค้ารายที่สอง** (นอกจาก Intel 18A) ประกาศใช้ High-NA ใน mass production → เทคโนโลยีเสาหลัก growth รอบหน้าไม่กระจาย พึ่ง Intel รายเดียวที่ประวัติ execution ไม่แน่นอน *(baseline วันนี้ = 1 ราย, Intel 18A)*

## 6. What to ask before owning it

- ถ้า order ปี 2027 booked เกือบหมดแล้ว (ตามที่ management บอก) การเติบโตปี 2028 จะมาจากไหน — ดีมานด์ใหม่จริง หรือแค่เลื่อนหัวคิว?

- China ~20% ของยอดขาย ถ้า export control เข้มขึ้นอีก กระทบตัวเลขปีหน้าเท่าไหร่ และมีลูกค้าอื่นมาชดเชยทันไหม?
- High-NA (เครื่องรุ่นใหม่แพงกว่า) จะกระจายไปลูกค้ารายที่สองเมื่อไหร่ หรือยังพึ่ง Intel รายเดียว?
